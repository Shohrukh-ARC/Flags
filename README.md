# Flags
Here I try to draw flags using HTML and CSS

All of the proporions (height-to-width ratio) and color hex codes (except the few that I did at the beginning) are taken from the internet.

Side note: For width and heght sometimes I write value by 8 more/less (e.g., instead of 300 I can write 292 or 308). 
The reason is default body margin, which can change height or width by 8 and that change is done considering this margin.
